# BizzyOTServer-14

BizzyOTServer-14 is a personal OpenTibiaBR Canary 14.x project that I am using to learn server management, website development, database integration, asset editing, and game customization. The project also serves as part of my software development portfolio as I continue working toward my degree.

## Current Features

* OpenTibiaBR Canary 14.x server
* MyAAC website integration
* In-game gift shop system
* Custom item support
* Animated item and shop images
* Custom travel modifications
* Gameplay rate adjustments
* Tibia 14 asset editing
* Discord integration
* Donation shop configuration
* Expanded webshop item catalog
* Tibia 14.05 client support
* Website customization and English localization

## Current Goals
