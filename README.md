# BizzyOTServer-14

BizzyOTServer-14 is a personal OpenTibiaBR Canary 14.x customization project focused on learning MMORPG server management, MyAAC website integration, custom item creation, asset editing, gameplay customization, and future hosting/security deployment.

## Current Features
- OpenTibiaBR Canary 14.x server
- MyAAC website integration
- Working in-game gift shop system
- Custom VIP item support
- Animated item/shop images
- Custom travel modifications
- Gameplay rate modifications
- Tibia 14 asset editing workflow

## Current Goals
- Additional VIP items
- Event systems
- Website improvements
- Donation/payment integration
- Hosting and security setup
- Public testing and long-term server management

## Technologies Used
- Lua
- XML
- PHP
- MySQL
- XAMPP
- OpenTibiaBR Canary
- Assets Editor
- GitHub
